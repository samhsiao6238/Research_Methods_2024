## 說明

1. 更正後項目總計相關性：這一欄顯示了每個題目與問卷其餘部分的相關程度。一般來說，這個值越高意味著該題目與整體問卷的一致性越好。

## 語法

_信度分析_

<br>

1. 開啟指定的資料集 `資料集1`。

    ```bash
    *========================================.
    DATASET ACTIVATE 資料集1.
    *========================================.
    ```

<br>

2. 社交焦慮-整體。

    ```bash
    *===============================================.
    *社交焦慮信度：完整19題的信度.

    RELIABILITY
      /VARIABLES=Ｑan01 TO Ｑan19
      /SCALE('ALL VARIABLES') ALL
      /MODEL=ALPHA
      /SUMMARY=TOTAL.
    *===============================================.
    ```

<br>

3. 嘗試進行刪除看看。

    <img src="images/img_05.png" width="400px">

<br>

4. 社交焦慮-刪除兩題 Qan08、Qan10 。
