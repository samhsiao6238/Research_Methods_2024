# 隱藏點讚

_`是否顯示點讚數` 這個變數具有兩個水平，問卷初始狀態是 `長格式`，所以以下將分做兩個部分進行，首先進行的事 `隱藏點讚數的各項評分`。_

<br>

## 確認工作

1. 開啟指定的資料集。

    ```bash
    *========================================.
    DATASET ACTIVATE 資料集1.
    *========================================.
    ```

<br>

2. 編輯或設定 `標籤、值標籤、小數位數、變數類型、對齊`。

    ```bash
    *========================================.
    *標籤名稱.
    VARIABLE LABELS  Hide01 '隱藏點讚01-意圖1'.
    VARIABLE LABELS  Hide02 '隱藏點讚02-意圖2'.
    VARIABLE LABELS  Hide03 '隱藏點讚03-意圖3'.
    VARIABLE LABELS  Hide04 '隱藏點讚04-數量1'.
    VARIABLE LABELS  Hide05 '隱藏點讚05-數量2'.
    VARIABLE LABELS  Hide06 '隱藏點讚06-數量3'.
    VARIABLE LABELS  Hide07 '隱藏點讚07-數量4'.
    VARIABLE LABELS  Hide08 '隱藏點讚08-正負面1'.
    VARIABLE LABELS  Hide09 '隱藏點讚09-正負面2'.
    VARIABLE LABELS  Hide10 '隱藏點讚10-誠實度1'.
    VARIABLE LABELS  Hide11 '隱藏點讚11-誠實度2'.
    VARIABLE LABELS  Hide12 '隱藏點讚12-誠實度3'.
    VARIABLE LABELS  Hide13 '隱藏點讚13-誠實度4'.
    VARIABLE LABELS  Hide14 '隱藏點讚14-深度1'.
    VARIABLE LABELS  Hide15 '隱藏點讚15-深度2'.
    VARIABLE LABELS  Hide16 '隱藏點讚16-深度3'.
    VARIABLE LABELS  Hide17 '隱藏點讚17-相關性1'.
    VARIABLE LABELS  Hide18 '隱藏點讚18-相關性2'.

    *小數的位數為「0」，F 為 Format 的意思.
    FORMATS Hide01 to Hide18 (F1.0).

    *靠左 置中 靠右 LEFT CENTER RIGHT.
    VARIABLE ALIGN Hide01 to Hide18(CENTER).

    *變數類型為序數.
    VARIABLE LEVEL Hide01 to Hide18 (ORDINAL).

    *值標籤.
    VALUE LABELS Hide01 to Hide18
    0 '其他'  
    1 '不會如此'  
    2 '很少如此'  
    3 '有時如此'  
    4 '經常如此'  
    5 '總是如此'. 
    *執行.
    EXECUTE.
    *========================================.
    ```

## 繪圖

1. 次數分配＋繪圖。

    ```bash
    *========================================.
    *次數分配＋繪圖：社會焦慮問題 Hide01 到 Hide18.
    FREQUENCIES VARIABLES=Hide01 TO Hide18
    /BARCHART FREQ
    /ORDER=ANALYSIS.
    EXECUTE.
    *========================================.
    ```
