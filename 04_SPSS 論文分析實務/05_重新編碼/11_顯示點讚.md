2. 標籤、值標籤、小數位數、變數類型、對齊。

    ```bash
    *========================================.
    *標籤名稱.
    VARIABLE LABELS  Show01 '顯示點讚01-意圖1'.
    VARIABLE LABELS  Show02 '顯示點讚02-意圖2'.
    VARIABLE LABELS  Show03 '顯示點讚03-意圖3'.
    VARIABLE LABELS  Show04 '顯示點讚04-數量1'.
    VARIABLE LABELS  Show05 '顯示點讚05-數量2'.
    VARIABLE LABELS  Show06 '顯示點讚06-數量3'.
    VARIABLE LABELS  Show07 '顯示點讚07-數量4'.
    VARIABLE LABELS  Show08 '顯示點讚08-正負面1'.
    VARIABLE LABELS  Show09 '顯示點讚09-正負面2'.
    VARIABLE LABELS  Show10 '顯示點讚10-誠實度1'.
    VARIABLE LABELS  Show11 '顯示點讚11-誠實度2'.
    VARIABLE LABELS  Show12 '顯示點讚12-誠實度3'.
    VARIABLE LABELS  Show13 '顯示點讚13-誠實度4'.
    VARIABLE LABELS  Show14 '顯示點讚14-深度1'.
    VARIABLE LABELS  Show15 '顯示點讚15-深度2'.
    VARIABLE LABELS  Show16 '顯示點讚16-深度3'.
    VARIABLE LABELS  Show17 '顯示點讚17-相關性1'.
    VARIABLE LABELS  Show18 '顯示點讚18-相關性2'.

    *小數的位數為「0」，F 為 Format 的意思.
    FORMATS Show01 to Show18 (F1.0).

    *靠左 置中 靠右 LEFT CENTER RIGHT.
    VARIABLE ALIGN Show01 to Show18(CENTER).

    *變數類型為序數.
    VARIABLE LEVEL Show01 to Show18 (ORDINAL).

    *值標籤.
    VALUE LABELS Show01 to Show18
    0 '其他'  
    1 '不會如此'  
    2 '很少如此'  
    3 '有時如此'  
    4 '經常如此'  
    5 '總是如此'. 
    *執行.
    EXECUTE.
    *========================================.
    ```
