# 使用數據

<br>

1. 開啟 SPSS，假如有製作快速啟動，可透過終端機執行以下指令。

    ```bash
    spss
    ```

<br>

2. 在 SPSS 中選取 `檔案 -> 新建 -> 語法`，然後運行以下指令。

    ```bash
    GET FILE='/Users/samhsiao/Desktop/_test02_/simulated_data_complete.sav'.
    DATASET NAME 資料集1.
    ```
