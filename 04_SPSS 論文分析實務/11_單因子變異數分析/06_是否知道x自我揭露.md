# 是否知道 x 自我揭露

<br>

## 使用語法

1. 隱藏。

    ```bash
    *========================================.
    *單變量變異數分析：是否知道功能x自我揭露：隱藏.
    UNIANOVA avgHide BY isKnowHideFunction
    /METHOD=SSTYPE(3)
    /INTERCEPT=INCLUDE
    /PRINT HOMOGENEITY
    /CRITERIA=ALPHA(.05)
    /DESIGN=isKnowHideFunction.
    *========================================.
    ```
